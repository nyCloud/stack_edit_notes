## CSS

### 1. Intro

A CSS rule is usually consisted by a selection and several css statements.
```css
/* selector {stmt1; stmt2}*/
h1 {color: blue; font-size: 12px;}
```

### 2. CSS Selector
#### 2.1 ID Selector 
```css
/* This style will apply to element with <id = 'id1'>  */
#id1 {
	text-align: center;
	color: red;
}
```
