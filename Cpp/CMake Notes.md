## 1. A Basic Starting Point 
```cpp
cmake_minimum_required(VERSION 3.10)

# Project name

# Specify Cpp standard
set(CMAKE_CXX_STANDARD 11)
set(CMAKE_CXX_STANDARD_REQUIRED True)

# Add the executable
add_executable(Tutorial tutorial.cpp)
```
